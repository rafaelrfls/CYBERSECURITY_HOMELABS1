Script para consulta de IPs utilizando a API do VirusTotal. 
 
A partir de um endereço IP informado, o script: 
- Consulta o IP no VirusTotal 
- Coleta a reputação do IP 
- Jarm 
- Owner do IP 
- País de origem e continente 
- Analisa estatísticas como: 
 Malicious 
 Suspicious 
 Harmless 
 Undetected 
- Data da última análise 
- Gera o link para acesso rápido ao site do VirusTotal 
 
Com esse script aprendi: 
- Consumo de API REST via Bash 
- Uso de curl e jq para parsing de JSON 
- Automação aplicada ao SOC N1
